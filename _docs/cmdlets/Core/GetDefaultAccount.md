## Syntax
```powershell
Get-AzureDevOpsStartUpAccount
```

<br>
## Description
Displays your configured start up account.  I.E. the account that is set as the current account context when the module is imported.

<br>
## Example
```powershell
Get-AzureDevOpsStartUpAccount
```

## Parameters
This Cmdlet has no input parameters
